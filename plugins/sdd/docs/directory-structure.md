# SDD Directory Structure

ユーザープロジェクトに生成されるファイル:

```
.claude/skills/steering/   # プロジェクト全体のステアリング（Progressive Disclosure）
├── SKILL.md               # ステアリングスキル本体
├── product.md             # プロダクト方針
├── tech.md                # 技術スタック
├── structure.md           # プロジェクト構造
└── principles.md          # 開発原則

specs/
├── research/            # 調査結果
├── {spec}/              # 仕様書ディレクトリ
│   ├── overview.md
│   ├── specification.md
│   ├── technical-details.md
│   └── phases/
└── _archived/           # アーカイブ済みspec
```
